How It Works

The reading progress feature automatically tracks your reading journey and provides visual feedback about your progress through the novel.

Visual Indicators

  • Progress Bar: Shows overall completion percentage with animated gradient fill
  • Current Chapter: Highlighted in purple with indicator line and chevron
  • Read Chapters: Dimmed text with green checkmark (✓)
  • Unread Chapters: Normal brightness with no indicator

Automatic Tracking

No manual action required! The system automatically:

  • Marks chapters as read when you view them
  • Updates progress when you scroll to the next chapter
  • Saves your progress to localStorage
  • Syncs progress across browser tabs
  • Restores progress when you return

Technical Implementation

// Reading Progress Store (Zustand + localStorage) interface ReadingProgressState { currentChapterId: number | null; readChapterIds: Set<number>; totalChapters: number; getProgressPercentage: () => number; } // Automatic tracking in ChapterView useEffect(() => { if (chapter) { setCurrentChapter(chapter.id); // Auto-mark as read } }, [chapter]);

Key Features

🎯 Accurate

Tracks every chapter you read with O(1) lookup performance

💾 Persistent

Progress saved to localStorage, never lose your place

🔄 Synced

Progress syncs across browser tabs automatically

⚡ Fast

Minimal re-renders, efficient Set-based storage
