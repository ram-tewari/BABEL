# Logs Directory

This directory contains log files generated by BABEL during operation.

## Log Files

- **`babel_transform.log`** - Phase 1 transformation logs (API calls, errors, retries)
- Other log files may be generated by different phases

## Configuration

Log level can be configured via environment variable:
```bash
BABEL_LOG_LEVEL=INFO  # DEBUG, INFO, WARNING, ERROR, CRITICAL
```

## Cleanup

Log files are automatically gitignored. To clean up old logs:
```bash
# Windows
del logs\*.log

# Unix
rm logs/*.log
```

## Log Format

Logs include:
- Timestamp
- Log level
- Module name
- Message
- Additional context (API responses, errors, etc.)
